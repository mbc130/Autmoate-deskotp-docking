# Autmoate-deskotp-docking
this is a python script 
FOR 
linux users
using appimages
wanting to integerate their applications into the dock as their favourites 
wanting to skip the desktop entry making and not wanting ot repeat this task 


## how to use it 

1. Download your AppImage (e.g., CoolApp.AppImage).
2. Run the script with the path to the AppImage:
        python3 install_appimage.py /path/to/CoolApp.AppImage
3. seach your app name .You can now add it to your favorites.
